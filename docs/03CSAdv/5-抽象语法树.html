д»Һжәҗд»Јз ҒпјҲз¬ҰеҸ·пјүдёҖи§Јжһҗе°ұеҫ—еҲ°дәҶжЁЎеһӢпјҢзЁӢеәҸиҜӯиЁҖеҶ…йғЁе®һйҷ…дёҠжҳҜз”ЁжҠҪиұЎиҜӯжі•ж ‘жқҘиЎЁзӨәзҡ„гҖӮзј–иҜ‘пјҢе°ұжҳҜеӨ„зҗҶиҝҷдёӘжҠҪиұЎиҜӯжі•ж ‘жЁЎеһӢпјҢдјҳеҢ–пјҢе№¶иҝӣиЎҢд»Јз Ғз”ҹжҲҗгҖӮ

жҠҪиұЎиҜӯжі•з®Җд»Ӣ вҖӢ

еҪ“зј–иҜ‘еҷЁеӨ„зҗҶжәҗд»Јз Ғж—¶пјҢе®ғдјҡйҖҡиҝҮиҜҚжі•еҲҶжһҗе’ҢиҜӯжі•еҲҶжһҗе°Ҷжәҗд»Јз ҒиҪ¬жҚўдёәжҠҪиұЎиҜӯжі•ж ‘пјҲAbstract Syntax TreeпјҢASTпјүгҖӮжҠҪиұЎиҜӯжі•ж ‘жҳҜдёҖз§Қд»Ҙж ‘еҪўз»“жһ„иЎЁзӨәд»Јз ҒиҜӯжі•з»“жһ„зҡ„жЁЎеһӢгҖӮ

жҠҪиұЎиҜӯжі•ж ‘жҳҜжәҗд»Јз Ғзҡ„жҠҪиұЎиЎЁзӨәеҪўејҸпјҢе®ғеҝҪз•ҘдәҶжәҗд»Јз Ғдёӯзҡ„дёҚеҝ…иҰҒз»ҶиҠӮпјҢеҸӘдҝқз•ҷдәҶиҜӯжі•з»“жһ„е’Ңе…ій”®дҝЎжҒҜгҖӮе®ғжҚ•жҚүдәҶжәҗд»Јз Ғзҡ„еұӮж¬Ўз»“жһ„гҖҒиЎЁиҫҫејҸзҡ„дјҳе…Ҳзә§е’Ңе…іиҒ”е…ізі»зӯүгҖӮ

еңЁжҠҪиұЎиҜӯжі•ж ‘дёӯпјҢжҜҸдёӘиҠӮзӮ№д»ЈиЎЁд»Јз Ғзҡ„дёҖдёӘиҜӯжі•з»“жһ„пјҢеҰӮиЎЁиҫҫејҸгҖҒиҜӯеҸҘгҖҒеҮҪж•°еЈ°жҳҺзӯүгҖӮиҠӮзӮ№д№Ӣй—ҙйҖҡиҝҮзҲ¶еӯҗе…ізі»е’Ңе…„ејҹе…ізі»зӣёиҝһжҺҘпјҢеҪўжҲҗдёҖжЈөж ‘зҡ„з»“жһ„гҖӮжҜҸдёӘиҠӮзӮ№йғҪеҢ…еҗ«жңүе…іиҜҘиҜӯжі•з»“жһ„зҡ„дҝЎжҒҜпјҢеҰӮиҝҗз®—з¬ҰгҖҒж“ҚдҪңж•°гҖҒеҸҳйҮҸеҗҚзӯүгҖӮ

зј–иҜ‘еҷЁеңЁеӨ„зҗҶжҠҪиұЎиҜӯжі•ж ‘ж—¶пјҢеҸҜд»ҘиҝӣиЎҢеҗ„з§ҚдјҳеҢ–е’ҢиҪ¬жҚўж“ҚдҪңгҖӮдҫӢеҰӮпјҢеҸҜд»ҘйҖҡиҝҮйқҷжҖҒеҲҶжһҗжқҘжЈҖжөӢжҪңеңЁзҡ„й”ҷиҜҜе’ҢдјҳеҢ–жңәдјҡпјҢиҝӣиЎҢеёёйҮҸжҠҳеҸ гҖҒеҫӘзҺҜеұ•ејҖгҖҒж— ж•Ҳд»Јз ҒеҲ йҷӨзӯүдјҳеҢ–пјҢд»ҘжҸҗй«ҳзЁӢеәҸзҡ„жҖ§иғҪе’Ңж•ҲзҺҮгҖӮжӯӨеӨ–пјҢзј–иҜ‘еҷЁиҝҳеҸҜд»Ҙж №жҚ®жҠҪиұЎиҜӯжі•ж ‘з”ҹжҲҗзӣ®ж Үд»Јз ҒпјҢе°Ҷй«ҳзә§иҜӯиЁҖд»Јз ҒиҪ¬жҚўдёәеҸҜжү§иЎҢзҡ„жңәеҷЁз ҒжҲ–еӯ—иҠӮз ҒгҖӮ

жҠҪиұЎиҜӯжі•ж ‘еңЁзј–иҜ‘еҷЁе’Ңи§ЈйҮҠеҷЁдёӯжү®жј”зқҖйҮҚиҰҒзҡ„и§’иүІпјҢе®ғжҸҗдҫӣдәҶдёҖз§ҚдҫҝдәҺеӨ„зҗҶе’ҢеҲҶжһҗд»Јз Ғзҡ„з»“жһ„еҢ–иЎЁзӨәеҪўејҸгҖӮйҖҡиҝҮдҪҝз”ЁжҠҪиұЎиҜӯжі•ж ‘пјҢзј–иҜ‘еҷЁеҸҜд»Ҙжӣҙе®№жҳ“ең°иҝӣиЎҢд»Јз ҒеҲҶжһҗгҖҒдјҳеҢ–е’Ңз”ҹжҲҗпјҢд»ҺиҖҢе®һзҺ°еҜ№зЁӢеәҸзҡ„жңүж•ҲеӨ„зҗҶгҖӮ

дёҖдёӘз®ҖеҚ•зҡ„cзҡ„жҠҪиұЎиҜӯжі•ж ‘дҫӢеӯҗ вҖӢ

дёӢйқўжҳҜдёҖдёӘжһҒз®ҖеҚ•зҡ„CиҜӯиЁҖд»Јз ҒзӨәдҫӢеҸҠе…¶еҜ№еә”зҡ„жҠҪиұЎиҜӯжі•ж ‘пјҡ

CиҜӯиЁҖд»Јз ҒзӨәдҫӢпјҡ

c
#include <stdio.h>

int main() {
    int a = 5;
    int b = 10;
    int sum = a + b;
    
    printf("The sum is: %d\n", sum);
    
    return 0;
}

еңЁиҝҷдёӘзӨәдҫӢдёӯпјҢжҠҪиұЎиҜӯжі•ж ‘д»Ҙж ‘зҡ„еҪўејҸеұ•зӨәдәҶд»Јз Ғзҡ„иҜӯжі•з»“жһ„гҖӮж ‘зҡ„ж №иҠӮзӮ№жҳҜProgramпјҢиЎЁзӨәж•ҙдёӘзЁӢеәҸгҖӮProgramиҠӮзӮ№дёӢйқўжҳҜдёҖдёӘCompoundиҠӮзӮ№пјҢиЎЁзӨәдёҖдёӘеӨҚеҗҲиҜӯеҸҘпјҲд»Јз Ғеқ—пјүгҖӮ

еңЁCompoundиҠӮзӮ№дёӢпјҢжңүдёүдёӘDeclиҠӮзӮ№пјҢеҲҶеҲ«иЎЁзӨәдёүдёӘеҸҳйҮҸзҡ„еЈ°жҳҺгҖӮжҜҸдёӘDeclиҠӮзӮ№дёӢйқўжҳҜдёҖдёӘintиҠӮзӮ№пјҢиЎЁзӨәеҸҳйҮҸзҡ„зұ»еһӢпјҢд»ҘеҸҠдёҖдёӘиөӢеҖјиЎЁиҫҫејҸиҠӮзӮ№пјҢиЎЁзӨәеҸҳйҮҸзҡ„еҲқе§ӢеҢ–гҖӮ

еңЁд»Јз ҒдёӯпјҢжңүдёҖдёӘеҠ жі•иҝҗз®—иЎЁиҫҫејҸa + bпјҢе®ғиў«иЎЁзӨәдёәдёҖдёӘAddиҠӮзӮ№пјҢе·Ұдҫ§жҳҜдёҖдёӘa=5зҡ„иөӢеҖјиЎЁиҫҫејҸиҠӮзӮ№пјҢеҸідҫ§жҳҜдёҖдёӘb=10зҡ„иөӢеҖјиЎЁиҫҫејҸиҠӮзӮ№гҖӮ

жңҖеҗҺпјҢд»Јз Ғдёӯзҡ„printfиҜӯеҸҘиў«иЎЁзӨәдёәдёҖдёӘprintfиҠӮзӮ№пјҢе®ғеҢ…еҗ«дёҖдёӘеӯ—з¬ҰдёІеӯ—йқўйҮҸ"The sum is: %d\n"е’ҢдёҖдёӘеҸҳйҮҸеј•з”ЁsumгҖӮ

дёҖдёӘз®ҖеҚ•зҡ„fortranзҡ„жҠҪиұЎиҜӯжі•ж ‘зҡ„дҫӢеӯҗ вҖӢ

дёӢйқўжҳҜдёҖдёӘз®ҖеҚ•зҡ„Fortranд»Јз ҒзӨәдҫӢеҸҠе…¶еҜ№еә”зҡ„жҠҪиұЎиҜӯжі•ж ‘пјҡ

еңЁиҝҷдёӘзӨәдҫӢдёӯпјҢжҠҪиұЎиҜӯжі•ж ‘д»Ҙж ‘зҡ„еҪўејҸеұ•зӨәдәҶFortranд»Јз Ғзҡ„иҜӯжі•з»“жһ„гҖӮж ‘зҡ„ж №иҠӮзӮ№жҳҜProgramпјҢиЎЁзӨәж•ҙдёӘзЁӢеәҸгҖӮProgramиҠӮзӮ№дёӢйқўжҳҜдёҖдёӘProgramNameиҠӮзӮ№пјҢиЎЁзӨәзЁӢеәҸзҡ„еҗҚз§°гҖӮ

еңЁProgramиҠӮзӮ№зҡ„дёӢйқўпјҢжңүдёҖдёӘBlockиҠӮзӮ№пјҢиЎЁзӨәзЁӢеәҸзҡ„д»Јз Ғеқ—гҖӮеңЁд»Јз Ғеқ—дёӯпјҢжңүдёҖдёӘDeclarationsиҠӮзӮ№пјҢиЎЁзӨәеҸҳйҮҸзҡ„еЈ°жҳҺгҖӮжҜҸдёӘDeclиҠӮзӮ№дёӢйқўжҳҜдёҖдёӘINTEGERиҠӮзӮ№пјҢиЎЁзӨәеҸҳйҮҸзҡ„зұ»еһӢгҖӮ

еңЁд»Јз ҒдёӯпјҢжңүдёүдёӘиөӢеҖјиҜӯеҸҘпјҢеҲҶеҲ«жҳҜa = 5гҖҒb = 10е’Ңsum = a + bгҖӮжҜҸдёӘиөӢеҖјиҜӯеҸҘиў«иЎЁзӨәдёәдёҖдёӘAssignиҠӮзӮ№пјҢе·Ұдҫ§жҳҜдёҖдёӘеҸҳйҮҸиҠӮзӮ№пјҢеҸідҫ§жҳҜдёҖдёӘиЎЁиҫҫејҸиҠӮзӮ№гҖӮеңЁAssignиҠӮзӮ№дёӯпјҢдҪҝз”ЁдәҶAddиҠӮзӮ№иЎЁзӨәеҠ жі•иҝҗз®—пјҢе·Ұдҫ§жҳҜеҸҳйҮҸиҠӮзӮ№aпјҢеҸідҫ§жҳҜеҸҳйҮҸиҠӮзӮ№bгҖӮ

жңҖеҗҺпјҢд»Јз Ғдёӯзҡ„WRITEиҜӯеҸҘиў«иЎЁзӨәдёәдёҖдёӘWriteиҠӮзӮ№пјҢе®ғеҢ…еҗ«дёҖдёӘж јејҸжҺ§еҲ¶жҸҸиҝ°з¬ҰпјҲ*иЎЁзӨәй»ҳи®Өж јејҸпјүе’ҢеӨҡдёӘиЎЁиҫҫејҸиҠӮзӮ№гҖӮеңЁиҝҷдёӘзӨәдҫӢдёӯпјҢWriteиҠӮзӮ№зҡ„дёӨдёӘиЎЁиҫҫејҸеҲҶеҲ«жҳҜеӯ—з¬ҰдёІеӯ—йқўйҮҸ'The sum is:'е’ҢеҸҳйҮҸиҠӮзӮ№sumгҖӮ

дәӨдә’ејҸFortranдёҺLFortran вҖӢ

жңүдёҖдёӘзҺ°д»Јfortranзҡ„е·Ҙе…·lfortranгҖӮеҸҜд»ҘдҪҝз”ЁlfotranжҹҘзңӢfortranзҡ„жҠҪиұЎиҜӯжі•ж ‘гҖӮ

LFortranжҳҜдёҖдёӘз”ЁдәҺзј–иҜ‘FortranиҜӯиЁҖзҡ„ејҖжәҗйЎ№зӣ®пјҢе®ғжҸҗдҫӣдәҶз”ҹжҲҗжҠҪиұЎиҜӯжі•ж ‘пјҲASTпјүзҡ„еҠҹиғҪгҖӮ

иҰҒдҪҝз”ЁLFortranжқҘжҹҘзңӢFortranд»Јз Ғзҡ„жҠҪиұЎиҜӯжі•ж ‘пјҢжӮЁйңҖиҰҒеңЁжң¬ең°е®үиЈ…LFortranе№¶дҪҝз”Ёе…¶е‘Ҫд»ӨиЎҢе·Ҙе…·гҖӮд»ҘдёӢжҳҜдёҖдёӘз®ҖеҚ•зҡ„жӯҘйӘӨзӨәдҫӢпјҡ

LFortranе°Ҷдјҡи§Јжһҗе№¶жҳҫзӨәFortranд»Јз Ғзҡ„жҠҪиұЎиҜӯжі•ж ‘гҖӮ

иҜ·жіЁж„ҸпјҢLFortranжҳҜдёҖдёӘејҖжәҗйЎ№зӣ®пјҢе…¶еҠҹиғҪе’Ңз”Ёжі•еҸҜиғҪдјҡжңүжүҖеҸҳеҢ–гҖӮе»әи®®жӮЁеҸӮиҖғLFortranзҡ„е®ҳж–№ж–ҮжЎЈе’Ңиө„жәҗд»ҘиҺ·еҫ—жӣҙиҜҰз»Ҷе’ҢжңҖж–°зҡ„дҝЎжҒҜгҖӮ

д»ҘдёӢжҳҜдёҖдёӘFortranд»Јз ҒзӨәдҫӢеҸҠе…¶еҜ№еә”зҡ„жҠҪиұЎиҜӯжі•ж ‘пјҲзұ»дјјLFortranз”ҹжҲҗзҡ„иҫ“еҮәпјүпјҡ

Fortranд»Јз ҒзӨәдҫӢпјҲexample.f90пјүпјҡ

fortran
PROGRAM HelloWorld
  IMPLICIT NONE
  
  INTEGER :: a, b, sum
  
  a = 5
  b = 10
  sum = a + b
  
  WRITE(*,*) 'The sum is:', sum
  
END PROGRAM HelloWorld

еңЁиҝҷдёӘзӨәдҫӢдёӯпјҢжҠҪиұЎиҜӯжі•ж ‘д»ҘдёҖз§Қзұ»дјјдәҺж ‘зҠ¶зҡ„ж–Үжң¬еҪўејҸе‘ҲзҺ°гҖӮе®ғеұ•зӨәдәҶд»Јз Ғзҡ„з»“жһ„е’Ңз»„з»Үж–№ејҸгҖӮжҜҸдёӘиҠӮзӮ№йғҪе…·жңүзӣёеә”зҡ„еұһжҖ§пјҢеҰӮеҸҳйҮҸеҗҚгҖҒзұ»еһӢгҖҒиЎЁиҫҫејҸзӯүгҖӮ

иҜ·жіЁж„ҸпјҢе®һйҷ…зҡ„LFortranиҫ“еҮәеҸҜиғҪдјҡж №жҚ®е…¶зүҲжң¬е’Ңй…ҚзҪ®иҖҢжңүжүҖдёҚеҗҢгҖӮд»ҘдёҠзӨәдҫӢд»…з”ЁдәҺиҜҙжҳҺжҠҪиұЎиҜӯжі•ж ‘зҡ„з»“жһ„е’ҢжҰӮеҝөпјҢе№¶йқһе®һйҷ…зҡ„LFortranиҫ“еҮәгҖӮеҰӮйңҖиҺ·еҸ–зңҹе®һзҡ„LFortranиҫ“еҮәпјҢиҜ·жҢүз…§еүҚйқўжҸҗеҲ°зҡ„жӯҘйӘӨеңЁжң¬ең°иҝҗиЎҢLFortranе‘Ҫд»ӨгҖӮ

жқҘдёӘJuliaзҡ„дҫӢеӯҗ вҖӢ

Juliaзј–зЁӢиҜӯиЁҖжң¬иә«жҸҗдҫӣдәҶеҶ…зҪ®зҡ„Meta.parseеҮҪж•°пјҢеҸҜз”ЁдәҺи§ЈжһҗJuliaд»Јз Ғе№¶з”ҹжҲҗжҠҪиұЎиҜӯжі•ж ‘пјҲASTпјүгҖӮ

д»ҘдёӢжҳҜдёҖдёӘзӨәдҫӢJuliaд»Јз ҒеҸҠе…¶еҜ№еә”зҡ„жҠҪиұЎиҜӯжі•ж ‘пјҲдҪҝз”ЁMeta.parseз”ҹжҲҗпјүпјҡ

Juliaд»Јз ҒзӨәдҫӢпјҡ

julia
function calculate_sum(a, b)
    sum = a + b
    println("The sum is: ", sum)
end

calculate_sum(5, 10)

дҪҝз”ЁMeta.parseз”ҹжҲҗзҡ„жҠҪиұЎиҜӯжі•ж ‘пјҡ

quote
    function calculate_sum(a, b)
        sum = a + b
        println("The sum is: ", sum)
    end
    calculate_sum(5, 10)
end

еңЁиҝҷдёӘзӨәдҫӢдёӯпјҢжҠҪиұЎиҜӯжі•ж ‘д»ҘJuliaд»Јз Ғзҡ„иЎЁиҫҫеҪўејҸе‘ҲзҺ°пјҢдҪҝз”Ёquoteе’ҢendеҢ…еӣҙд»Јз Ғеқ—гҖӮquoteе’Ңendд№Ӣй—ҙжҳҜж•ҙдёӘд»Јз Ғеқ—зҡ„жҠҪиұЎиҜӯжі•ж ‘иЎЁзӨәгҖӮ

еңЁд»Јз Ғеқ—дёӯпјҢжңүдёҖдёӘfunctionиҠӮзӮ№пјҢиЎЁзӨәеҮҪж•°е®ҡд№үгҖӮеҮҪж•°е®ҡд№үеҢ…еҗ«еҮҪж•°еҗҚз§°гҖҒеҸӮж•°еҲ—иЎЁе’ҢеҮҪж•°дҪ“гҖӮеҮҪж•°дҪ“еҶ…йғЁжңүдёҖдёӘиөӢеҖјиҜӯеҸҘиҠӮзӮ№sum = a + bе’ҢдёҖдёӘprintlnиҜӯеҸҘиҠӮзӮ№пјҢеҲҶеҲ«иЎЁзӨәеҸҳйҮҸиөӢеҖје’Ңиҫ“еҮәиҜӯеҸҘгҖӮ

жңҖеҗҺпјҢеңЁд»Јз Ғеқ—еӨ–йғЁи°ғз”ЁдәҶcalculate_sumеҮҪж•°е№¶дј йҖ’дәҶеҸӮж•°5е’Ң10пјҢиЎЁзӨәеҮҪж•°и°ғз”Ёзҡ„жҠҪиұЎиҜӯжі•ж ‘иҠӮзӮ№гҖӮ

иҜ·жіЁж„ҸпјҢе®һйҷ…з”ҹжҲҗе’Ңи§ЈжһҗJuliaд»Јз Ғзҡ„жҠҪиұЎиҜӯжі•ж ‘еҸҜиғҪйңҖиҰҒдҪҝз”ЁJuliaзј–зЁӢзҺҜеўғжҲ–зӣёеә”зҡ„е·Ҙе…·еә“гҖӮд»ҘдёҠзӨәдҫӢд»…з”ЁдәҺиҜҙжҳҺжҠҪиұЎиҜӯжі•ж ‘зҡ„з»“жһ„е’ҢжҰӮеҝөпјҢе№¶йқһе®һйҷ…зҡ„JuliaжҠҪиұЎиҜӯжі•ж ‘гҖӮеҰӮйңҖиҺ·еҸ–зңҹе®һзҡ„JuliaжҠҪиұЎиҜӯжі•ж ‘пјҢиҜ·еҸӮиҖғJuliaе®ҳж–№ж–ҮжЎЈе’Ңзӣёе…іиө„жәҗгҖӮ

жҠҪиұЎиҜӯжі•ж ‘дёҺжЁЎеһӢ вҖӢ

жҠҪиұЎиҜӯжі•ж ‘е°ұжҳҜжҲ‘д»¬жәҗд»Јз Ғз¬ҰеҸ·жүҖеҜ№еә”зҡ„жЁЎеһӢгҖӮз»ҸиҝҮparserд№ӢеҗҺпјҢе°ұжҳҜеӨ„зҗҶжҲ‘д»¬зҡ„жЁЎеһӢдәҶгҖӮ

жҠҪиұЎиҜӯжі•ж ‘пјҲASTпјүеҸҜд»ҘзңӢдҪңжҳҜе°Ҷжәҗд»Јз ҒиҪ¬жҚўдёәдёҖз§Қз»“жһ„еҢ–зҡ„жЁЎеһӢгҖӮйҖҡиҝҮи§ЈжһҗеҷЁпјҲParserпјүеҜ№жәҗд»Јз ҒиҝӣиЎҢиҜҚжі•еҲҶжһҗе’ҢиҜӯжі•еҲҶжһҗпјҢеҸҜд»Ҙжһ„е»әеҮәжҠҪиұЎиҜӯжі•ж ‘гҖӮ

жҠҪиұЎиҜӯжі•ж ‘жҳҜжәҗд»Јз Ғзҡ„дёҖз§ҚжҠҪиұЎиЎЁзӨәеҪўејҸпјҢе®ғеҝҪз•ҘдәҶжәҗд»Јз Ғдёӯзҡ„дёҚеҝ…иҰҒз»ҶиҠӮпјҢеҸӘдҝқз•ҷдәҶиҜӯжі•з»“жһ„е’Ңе…ій”®дҝЎжҒҜгҖӮе®ғжҚ•жҚүдәҶжәҗд»Јз Ғзҡ„еұӮж¬Ўз»“жһ„гҖҒиЎЁиҫҫејҸзҡ„дјҳе…Ҳзә§е’Ңе…іиҒ”е…ізі»зӯүгҖӮ

дёҖж—ҰжҲ‘д»¬жңүдәҶжҠҪиұЎиҜӯжі•ж ‘пјҢе°ұеҸҜд»ҘеҜ№е…¶иҝӣиЎҢеӨ„зҗҶе’ҢеҲҶжһҗпјҢиҝӣиҖҢиҝӣиЎҢзј–иҜ‘гҖҒдјҳеҢ–е’Ңд»Јз Ғз”ҹжҲҗзӯүж“ҚдҪңгҖӮ

зј–иҜ‘еҷЁйҖҡеёёдјҡеҜ№жҠҪиұЎиҜӯжі•ж ‘иҝӣиЎҢеҗ„з§ҚеӨ„зҗҶпјҢдҫӢеҰӮйқҷжҖҒеҲҶжһҗгҖҒдјҳеҢ–е’Ңд»Јз Ғз”ҹжҲҗгҖӮйқҷжҖҒеҲҶжһҗеҸҜд»ҘжЈҖжөӢжҪңеңЁзҡ„й”ҷиҜҜгҖҒдјҳеҢ–жңәдјҡе’Ңд»Јз ҒиҙЁйҮҸй—®йўҳгҖӮдјҳеҢ–ж¶үеҸҠеҜ№жҠҪиұЎиҜӯжі•ж ‘иҝӣиЎҢеҗ„з§ҚдјҳеҢ–жҠҖжңҜпјҢдҫӢеҰӮеёёйҮҸжҠҳеҸ гҖҒеҫӘзҺҜеұ•ејҖе’Ңж— ж•Ҳд»Јз ҒеҲ йҷӨпјҢд»ҘжҸҗй«ҳзЁӢеәҸзҡ„жҖ§иғҪе’Ңж•ҲзҺҮгҖӮд»Јз Ғз”ҹжҲҗйҳ¶ж®өе°ҶжҠҪиұЎиҜӯжі•ж ‘иҪ¬жҚўдёәзӣ®ж Үд»Јз ҒпјҢз”ҹжҲҗеҸҜжү§иЎҢзҡ„жңәеҷЁз ҒжҲ–еӯ—иҠӮз ҒгҖӮ

йҖҡиҝҮдҪҝз”ЁжҠҪиұЎиҜӯжі•ж ‘пјҢзј–иҜ‘еҷЁеҸҜд»Ҙжӣҙж–№дҫҝең°еӨ„зҗҶе’ҢеҲҶжһҗд»Јз ҒпјҢеӣ дёәжҠҪиұЎиҜӯжі•ж ‘жҸҗдҫӣдәҶдёҖз§Қз»“жһ„еҢ–зҡ„иЎЁзӨәеҪўејҸгҖӮе®ғе°Ҷжәҗд»Јз ҒиҪ¬жҚўдёәжҳ“дәҺеӨ„зҗҶзҡ„жЁЎеһӢпјҢдҪҝеҫ—зј–иҜ‘еҷЁиғҪеӨҹиҝӣиЎҢйқҷжҖҒеҲҶжһҗгҖҒдјҳеҢ–е’Ңд»Јз Ғз”ҹжҲҗзӯүж“ҚдҪңпјҢд»Ҙе®һзҺ°еҜ№зЁӢеәҸзҡ„жңүж•ҲеӨ„зҗҶгҖӮ
