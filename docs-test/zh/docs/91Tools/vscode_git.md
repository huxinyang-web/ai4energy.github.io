## 生成公钥

```powershell
ssh-keygen -t rsa -C "xxxxxx@qq.com"
```

一路回车，有y/n则y。在红框中的文件夹中去找公钥文件。

![在这里插入图片描述]

**id_rsa是私钥，id_rsa.pub是公钥**

![在这里插入图片描述]

使用公钥私钥匹配就可以免去登录验证
